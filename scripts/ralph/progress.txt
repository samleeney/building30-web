## Codebase Patterns
- **Tailwind v4**: Use `@import "tailwindcss"` in CSS, `@tailwindcss/vite` plugin in vite.config.ts (not PostCSS)
- **CardDetail pattern**: `CardDetail` takes `cardId` + optional `onClose`, fetches via `useCard(id)`, inner component handles editing state. Use `useUpdateCard` with mods for tag add/remove (`+tag:name`, `-tag:name`) and type changes (`type:inbox`)
- **Vite v7**: Downgraded from v8 scaffold due to `@tailwindcss/vite` peer dep requiring `vite@^5 || ^6 || ^7`
- **Auth0**: Configured in main.tsx via `Auth0Provider` wrapping entire app, uses `VITE_` env vars
- **TanStack Query**: `QueryClientProvider` wrapping app in main.tsx
- **Env vars**: `.env.local` (gitignored via `*.local`), `.env.example` committed
- **API client**: `createApiClient(getToken)` factory in `src/lib/api-client.ts` ‚Äî pass `getAccessTokenSilently` from Auth0; returns `{get, post, put, delete}` methods
- **Types**: All shared API types in `src/lib/types.ts` ‚Äî Card (summary), CardDetail (full), ProjectContext, ServerEvent, TagStat, ApiError, PaginatedResponse
- **Hooks pattern**: `useApiClient()` hook in `src/lib/hooks/use-api-client.ts` returns memoized API client; all query/mutation hooks use it
- **Query keys**: `["cards", params]` for lists, `["card", id]` for detail, `["views", name]` for view endpoints, `["tags"]` for tags
- **Mutations**: All card mutations invalidate `["cards"]` and `["views"]`; detail mutations also invalidate `["card", id]`
- **SSE**: `useEventStream()` connects when authenticated, invalidates queries on card_upsert/card_delete events
- **Routing**: `react-router-dom` with `BrowserRouter` in main.tsx, `Routes`/`Route` in App.tsx, `NavLink` for sidebar active state
- **Layout**: Three-panel layout in `App.tsx` via `Layout` component with `<Outlet />` ‚Äî sidebar (w-48) + main content + optional detail panel
- **C3 Design**: Geist Sans/Mono via CDN `@font-face`, oklch monochrome tokens in `@theme` block, `* { border-radius: 0 !important }`, `.grid-bg` class for grid lines
- **Page pattern**: Each page in `src/pages/{Name}View.tsx`, exported as named export `{Name}View`
- **Sidebar nav**: `src/components/layout/Sidebar.tsx` with `NavLink` array ‚Äî active state uses `bg-accent text-white`
- **CardList/CardItem**: `src/components/cards/` ‚Äî `CardList` takes `cards`, `selectedId`, `onSelect`, `emptyMessage` props; `CardItem` shows type badge, title, tags, due date. Selection managed via `onSelect` callback (pages use `useSearchParams` for `?card=id`)
- **Type badges**: `typeColors` map in `CardItem.tsx` ‚Äî oklch monochrome badges per card_type
